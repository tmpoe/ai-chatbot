# @ai-chatbot/shared-types

Shared TypeScript types for the AI Chatbot monorepo.

## Usage

```typescript
import { ChatMessage, ChatRequest, ModelInfo } from '@ai-chatbot/shared-types';
```

## Types

- `ChatMessage` - Chat message structure
- `ChatRequest` / `ChatResponse` - API request/response types
- `ToolCall` - Tool calling types
- `ModelInfo` - Model metadata
- `ApiError` / `ApiResponse<T>` - Standard API response wrapper
