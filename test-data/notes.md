# Project Notes

## What is MCP?
Model Context Protocol (MCP) is a protocol that enables AI assistants to connect to external data sources and tools.

## Architecture
- **Frontend**: Next.js with TypeScript
- **Backend**: Python with FastAPI
- **MCP Server**: Filesystem server for file access
- **LLM**: Google Gemini

## Key Features
- ✅ Streaming chat responses
- ✅ Tool calling via MCP
- ✅ File system access
- ✅ Model selection in UI
- ✅ Clean architecture
